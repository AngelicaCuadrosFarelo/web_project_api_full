# Tripleten web_project_around_auth

# Los elementos clave de esta división son el registro de usuarios, la autorización y la autenticación.

Se realizó:

- Autenticación y autorización: los fundamentos absolutos necesarios para crear aplicaciones totalmente funcionales.

- Autenticación front-end con React: se realizó una autorización de usuarios en el front-end, por medio del token adquirido al registrarse como usuario, implementamos el localStorage.

* La seguridad de las aplicaciones web: una introducción a las posibles vulnerabilidades de la página web, pormedio de Regex, validacion de datos.

- Trabajar con el almacenamiento de datos del navegador.
